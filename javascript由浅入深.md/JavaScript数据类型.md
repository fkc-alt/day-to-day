# JavaScript数据类型

### JavaScript数据类型（七种）  

* `Null`  
* `Undefined`  
* `String`  
* `Object`  
* `Array`  
* `Number`  
* `Symbol`

> 在es10中加入了第七种原始类型`BigInt`，现已被最新Chrome支持  

[BigInt MDN](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/BigInt)

### 原始数据类型（六种）  

* `Null`  
* `Undefined`  
* `String`  
* `Array`  
* `Number`  
* `Symbol`

### 引用数据类型（一种）  

* `Object`
