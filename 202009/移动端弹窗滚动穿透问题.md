# 移动端弹窗滚动穿透问题

### 最简单处理

``` javascript
html,body{
  overflow:hidden
}
```

当弹出出现的时候,设置body的position为fixed,并设置overflow为hidden
```javascript
document.body.style.position = 'fixed';
document.body.style.overflow = 'hidden';
```

当弹层取消
```javascript
document.body.style.position = 'relative';
document.body.style.overflow = 'auto';
```
